# <span style="color:blue">Math Formulas</span>

## Описание проекта
Это проект с функциями для вычисления периметра и площадей фигур,написанный на Python

## Формулы

### Площадь (Area)
- **Circle**: S = πR²
- **Rectangle**: S = ab
- **Square**: S = a²
- **Triangle**: S = ah/2

### Периметр (Perimeter)
- **Circle**: P = 2πR
- **Rectangle**: P = 2a + 2b
- **Square**: P = 4a
- **Triangle**: P = a + b + c

## Описание функций

### Функции для вычисления площади

#### Circle:
На вход поступает число r, возвращает r^2*pi

    Пример:
        Ввод:
        r=2
        Вывод:
        area=12.56
#### Rectangle:
На вход поступают числа a и b, возвращает произведение a и b

    Пример:
        Ввод:
        a=2
        b=5
        Вывод:
        area=10
#### Square:
На вход поступает число a, возвращает a в квадрате

    Пример:
        Ввод:
        a=7
        Вывод:
        area=49

#### Triangle:
 На вход поступают числа a и h, возвращает a умноженное на половину h

    Пример:
        Ввод:
        a=5
        h=4
        Вывод:
        area=1

### Функции для вычисления периметра

#### Circle:
На вход поступает число r, возвращает r*2*pi

    Пример:
        Ввод:
        r=4
        Вывод:
        perimeter=25.132

#### Rectangle:
На вход поступают числа a и b, возвращает удвоенное произведение a и b

    Пример:
        Ввод:
        a=5
        b=7
        Вывод:
        perimeter=24

#### Square:
На вход поступает число a, возвращает a умноженное на 4

    Пример:
        Ввод:
        a=2
        Вывод:
        perimeter=8

#### Triangle: 
На вход поступают числа a, b и c, возвращает сумму этих чисел

    Пример:
        Ввод:
        a=2
        b=3
        c=4
        Вывод:
        perimeter=9
